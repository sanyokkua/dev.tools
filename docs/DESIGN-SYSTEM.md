### Backward-Compat Aliases

`colors.scss` defines backward-compat aliases (e.g. `--primary-color: var(--primary)`, `--surface-color: var(--surface)`) and utility color classes `.color-<name>` generated by a `@each` loop over `$md-colors`.

**Do not use these aliases in new code.** Use the design tokens directly.

## 2. Primitive Classes

Source: `src/styles/primitives.scss`. Imported once in `src/pages/_app.tsx` (right after `colors.scss`).

Do not redefine `.card`, `.chip`, or `.code-block` here — they live in `surfaces.scss` and `chip.scss`.

### Buttons

All buttons share base styles: `font: inherit; font-weight: 600; border-radius: var(--r-sm); padding: 9px 16px; display: inline-flex; align-items: center; gap: 8px; transition: 0.15s`

| Class           | Alias              | Description                     |
| --------------- | ------------------ | ------------------------------- |
| `.btn.primary`  | `.button-solid`    | Filled primary color, shadow    |
| `.btn.tonal`    | `.button-filled`   | Primary-container background    |
| `.btn.outline`  | `.button-outlined` | Transparent bg, outline border  |
| `.btn.ghost`    | `.button-ghost`    | Transparent, muted text         |
| `.btn.danger`   | `.button-danger`   | Error-colored border and text   |
| `.btn.sm`       | `.button-small`    | Reduced padding + 12.5px font   |
| `.btn:disabled` | `.button-disabled` | 0.5 opacity, not-allowed cursor |

### Editor Chrome

| Class             | Description                                                                               |
| ----------------- | ----------------------------------------------------------------------------------------- |
| `.editorpane`     | Card-styled container, flex column, `overflow: hidden`                                    |

## 4. Rules

1. **No CSS Modules** — all stylesheets are global, imported once in `_app.tsx`.
2. **Primitives first** — use `.btn`, `.field`, `.input`, `.pill`, etc. before writing custom styles.
3. **Tokens always** — use CSS custom properties (`var(--primary)`, `var(--card)`) not hardcoded hex values.
4. **Do not redefine** `.card`, `.chip`, `.code-block` in `primitives.scss` — they live in `surfaces.scss` / `chip.scss`.
5. **New page styles** — add `src/styles/<page>.scss` and import it in `_app.tsx`.
