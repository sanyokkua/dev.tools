# Description
Transform a real photo into an anime / cartoon illustration while keeping the same subject(s), pose, and composition recognizable. Use this when you have a real photograph and want a stylized anime or cartoon version of it that still clearly depicts the same scene and people.

# Prompt
This is a real photograph. Re-draw it as a high-quality anime / cartoon illustration, as if it were hand-drawn and digitally colored in a modern Japanese animation style. Transform the medium fully into illustration: clean line art, smooth cel-shaded coloring, expressive anime-style eyes and features, simplified but appealing shapes, and soft stylized lighting and shadows. Keep the same subject or subjects, the same pose, gesture, head angle, and facial expression, the same clothing and notable accessories, the same number of people, and the same overall composition, framing, and background layout so the scene stays clearly recognizable as the original photo. You may stylize faces, hair, skin, and background into the anime aesthetic, but keep each person recognizable — preserve their general likeness, hairstyle, hair color, and outfit. Render a polished, vibrant illustration with appealing colors and clean shading. Output the image at the same aspect ratio as the uploaded photo.

# Parameters
