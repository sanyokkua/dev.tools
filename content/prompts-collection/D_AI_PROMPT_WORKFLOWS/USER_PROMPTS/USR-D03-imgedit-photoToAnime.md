# Prompt ID
USR-D03-imgedit-photoToAnime

# Domain / Category
D — AI & Prompt Workflows / D03 Image Editing

# Description
Single-shot image-editing prompt that converts a real photo into an anime/cartoon style while keeping the subject and composition recognizable. Attach the source photo.

# Prompt
Convert the photo in this image into an anime/cartoon style. Keep the subject(s), pose, key features, and overall composition recognizable, but fully restyle the rendering into clean anime/cartoon art — stylized linework, cel-style shading, simplified but characterful features, and appropriate color treatment. This is a deliberate creative transformation (high stylistic change is expected), so the medium changes from photographic to illustrated while the scene stays identifiable. Maintain a coherent, high-quality result.

(Attach the source photo. Restyle — identifiable subject, medium changes to anime/cartoon.)

# Parameters
None — fixed-content, single-shot prompt. Input is the attached image.

# Example Values
N/A — attach a real photo to convert to anime/cartoon.
