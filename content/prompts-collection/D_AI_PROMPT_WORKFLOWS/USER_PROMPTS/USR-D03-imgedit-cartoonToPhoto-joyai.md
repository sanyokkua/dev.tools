# Prompt ID
USR-D03-imgedit-cartoonToPhoto-joyai

# Domain / Category
D — AI & Prompt Workflows / D03 Image Editing (JoyAI Image Edit)

# Description
Convert a cartoon or anime illustration into a photorealistic image while preserving the pose, outfit, and character. Use when you have an anime/cartoon drawing and want a believable real-photo version of the same character and scene.

# Prompt
Transform the cartoon or anime illustration in this image into a photorealistic photograph while preserving the character, pose, and composition. Re-render the subject as a real person (or real object) with natural, lifelike detail: realistic skin with visible pores and natural skin tones, real individual strands of hair, true-to-life fabric and material texture on the clothing, and physically accurate lighting, shadows, and reflections. Convert flat cel shading and line art into real three-dimensional form with believable depth and a natural depth of field. Keep the same character: preserve the pose, body position, gesture, gaze direction, expression, hairstyle, outfit, key colors, and the overall framing and composition so the photo is clearly the real-world version of the same character and scene. Translate the original mood and atmosphere into realistic lighting. The result should look like a genuine high-quality photograph, not a drawing or a 3D render. Keep the pose, outfit, and character preserved.
