# Prompt ID
USR-D03-imgedit-cartoonToPhoto-gptImage

# Domain / Category
D — AI & Prompt Workflows / D03 Image Editing (GPT Image 2 / 1.5)

# Description
Transform a cartoon or anime image into a photorealistic real photograph while preserving the pose, outfit, and character identity. Use this when you have a cartoon, anime, or illustrated character and want a believable real-life photographic version of the same character and scene.

# Prompt
This is a cartoon / anime illustration. Re-render it as a photorealistic real photograph, as if the same character and scene were captured with a professional camera in real life. Convert the medium fully from drawing to photography: replace flat or cel-shaded coloring with realistic three-dimensional lighting, real skin with natural texture and visible pores, real hair with individual strands, and real fabric with believable weave, folds, and material detail. Keep the same character identity, the same pose and gesture, the same facial expression, the same hairstyle and hair color, the same outfit, accessories, and the same composition, framing, and background layout so it is clearly the same character and scene made real. Translate stylized anime proportions into natural, realistic human proportions while keeping the character recognizable. Use natural, physically believable lighting and shadows and realistic, true-to-life colors and skin tones. Output a photorealistic, high-resolution photograph at the same aspect ratio as the uploaded image, with professional-camera quality and no remaining cartoon or illustrated look.

# Parameters
None — fixed-content, single-shot prompt. Input is the attached image (no text parameters).
