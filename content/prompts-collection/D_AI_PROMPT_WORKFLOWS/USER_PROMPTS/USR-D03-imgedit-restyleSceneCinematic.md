# Prompt ID
USR-D03-imgedit-restyleSceneCinematic

# Domain / Category
D — AI & Prompt Workflows / D03 Image Editing

# Description
Single-shot image-editing prompt that gives a nature/city photo a cinematic / pro-camera re-grade, preserving the scene's content while changing the look. Attach the source photo.

# Prompt
Re-render the nature/city photograph in this image with a cinematic, professional-camera look. Keep the scene's content, structures, layout, and composition recognizable and unchanged; you MAY restyle the lighting, color grade, contrast, atmosphere, and depth to a filmic, high-end look (rich tonal depth, cinematic color, controlled highlights/shadows, subtle atmosphere). Keep it photorealistic — do not add, remove, or rearrange elements, and do not turn it into an illustration. This is a creative re-grade of the photographic treatment, not a change of the scene.

(Attach the source photo. Restyle — content locked, look may change.)

# Parameters
None — fixed-content, single-shot prompt. Input is the attached image.

# Example Values
N/A — attach a nature or city photo to re-grade cinematically.
