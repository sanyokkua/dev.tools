# Prompt ID
USR-D03-imgedit-photoToAnime-joyai

# Domain / Category
D — AI & Prompt Workflows / D03 Image Editing (JoyAI Image Edit)

# Description
Convert a real photo into an anime/cartoon illustration while keeping the subject, pose, and composition recognizable. Use when you want a stylized anime or cartoon version of a real photograph that still clearly reads as the same scene.

# Prompt
Transform the real photograph in this image into a polished anime and cartoon illustration while keeping the subject, pose, and composition clearly recognizable. Re-render everything in a clean hand-drawn anime style: bold confident line art, smooth cel-shaded coloring with soft gradients, expressive stylized eyes, simplified yet characterful features, and vibrant harmonious colors. Stylize the background to match the same anime aesthetic. Preserve the layout faithfully — keep the same subject(s), their pose, body position, gesture, gaze direction, clothing, key colors, and the overall framing and composition the same as in the photo, so the stylized version is instantly identifiable as the same scene. Capture the original mood and atmosphere (calm, energetic, dramatic, cheerful) in the anime rendering. The result should look like a high-quality anime artwork of the same scene, not a filtered photo. Keep the subject and composition recognizable.
