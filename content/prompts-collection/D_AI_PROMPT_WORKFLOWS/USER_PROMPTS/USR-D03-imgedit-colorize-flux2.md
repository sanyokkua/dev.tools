# Description
Add natural, realistic, era-appropriate color to a black-and-white photo with all detail intact. Use when you have a grayscale or black-and-white photo of people, scenes, or cityscapes.

# Prompt
Colorize the black-and-white image in this image with natural, realistic, era-appropriate color, giving believable skin tones, coherent clothing colors, and a consistent environment and lighting. Keep the color subtle and photographic, matching the period of the photo. Preserve the exact same faces, expressions, composition, framing, and all existing detail and texture, changing only the color and adding nothing new to the scene.
