# Prompt ID
USR-D03-imgedit-cartoonToPhoto

# Domain / Category
D — AI & Prompt Workflows / D03 Image Editing

# Description
Single-shot image-editing prompt that converts a cartoon/anime image into a realistic photo while keeping the subject and composition recognizable. Attach the source image.

# Prompt
Convert the cartoon/anime image into a realistic photograph. Keep the subject(s), pose, key features, and overall composition recognizable, but fully re-render them as photorealistic — believable materials, skin/texture, lighting, and depth. Translate stylized features into plausible real-world equivalents while staying faithful to the original character/scene design. This is a deliberate creative transformation (medium changes from illustrated to photographic). Maintain a coherent, high-quality, realistic result; do not add or remove major elements.

(Attach the source cartoon/anime image. Restyle — identifiable subject, medium changes to photoreal.)

# Parameters
None — fixed-content, single-shot prompt. Input is the attached image.

# Example Values
N/A — attach a cartoon/anime image to convert to photoreal.
